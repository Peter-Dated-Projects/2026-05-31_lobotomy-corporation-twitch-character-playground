Implement layer L5 (emotion) of the agent-behavior model — the final layer. Read .charm/kb/architecture/agent-behavior-model.md (L5) and the concrete code in docs/research/npc-emotion-models.md (sections 2.1-2.5, including the _update_emotion sketch and default constants). Authoritative.

character.py: add continuous valence in [-1,1] and arousal in [0,1], plus seeded-per-username susceptibility/expressiveness (derive deterministically, same md5 approach). Add _update_emotion(dt, neighbors) called EVERY frame before the mode branch (emotion is orthogonal to Mode): exponential decay toward neutral (the legibility mechanic — decay must outrun contagion or panic latches forever), proximity-weighted contagion pulling toward neighbours' (arousal,valence) scaled by susceptibility*expressiveness, and a mild crowding arousal bump; clamp. Add emotion_face()->'default'|'battle'|'panic' quantization WITH hysteresis (enter/exit thresholds or hold-time) so faces don't strobe. Emotion modulates movement: arousal scales walk speed up (AROUSAL_SPEED_GAIN) and restlessness (JUMP up/IDLE down); valence scales sociability/separation (distressed wants more space).

PLUMBING: extend the neighbor records from L3 to also carry (arousal, valence, expressiveness) so contagion reuses the one shared neighbor scan — no new spatial query. Update world.py accordingly.

RENDERER (the one cross-file change emotion forces): compose_character already accepts emotion, but SpriteSets are pre-rendered at 'default' only. Make the provider build all THREE emotion face variants per clip (recommended: pre-render at load; ~3x face memory, fine at our scale) and extend SpriteSet.clip + Character.surface to pick frames by emotion_face(). Keep the graceful placeholder fallback intact.

chat/commands.py + world command routing: emotion impulses from commands (e.g. !hug nudges valence+ on both and keeps its EMOTING clip; add !panic -> arousal up/valence down and !cheer -> both up if cheap). Treat a command as an appraised event applying an instantaneous clamped impulse. Add settings: EMOTION_DECAY_PER_SEC, CONTAGION_RATE, CONTAGION_RADIUS, AROUSAL_SPEED_GAIN, CROWD_AROUSAL (start from the brief's table) + face hysteresis thresholds.

Tests: new tests/test_emotion.py (decay returns to neutral; contagion spreads a spike to a near neighbour with a lag; emotion_face quantization + hysteresis; movement modulation); ensure full suite incl. the asset-absent fallback path stays green. `uv run --project . pytest` green. Touch only the listed files.
