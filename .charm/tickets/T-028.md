Implement layer L4 (personality) of the agent-behavior model. Read .charm/kb/architecture/agent-behavior-model.md (L4) and the concrete recommendations + code sketches in docs/research/crowd-following-personality.md (sections 2.1-2.4). Authoritative.

NEW MODULE sim/personality.py (pure data, no pygame): frozen Personality dataclass with sociability/independence/restlessness floats in [0,1]; personality_for(username) deriving them deterministically by slicing byte-windows out of md5(f"persona:{username}") — IMPORTANT: salt it differently than assets/character_defs.assign_character so trait != appearance bucket. Optionally skew the distribution so true loners are a minority (the brief explains the distribution-shape matters more than the mean). Document determinism.

character.py: give Character a persona (passed in by World.spawn, mirroring how sprites/nameplate are injected, to keep it testable). Add a low-frequency decide-timer (a few Hz, not per-frame). While WANDER+grounded, run a trait-biased JOIN check: utility score = sociability * proximity_curve(d) * threshold_curve(n, persona) * (1-independence), where the per-character join count T = 1 + round(independence*MAX_JOIN_THRESHOLD) realizes Granovetter; fire into the EXISTING grouping flow when score clears an ENTER threshold, then commit for a min dwell. While GROUPED, run the symmetric LEAVE check (independence*restlessness*shrink_curve*dwell_curve) with a leave-count floor strictly below T (dual-threshold dead-band) + min dwell. Multiply JUMP_CHANCE/IDLE_CHANCE and facing-reroll cadence by restlessness. Chat commands stay AUTHORITATIVE: set a manual-hold timer on touch()/explicit commands that suppresses the autonomous check briefly so the AI never immediately undoes a viewer's !follow/!leave.

world.py: expose the nearest reachable cluster (size+distance) for the join check (reuse grouping data; only count clusters on a reachable surface), pass persona into Character at spawn, and route autonomous joins through the existing _add_to_group / set_wander machinery — no new Mode, no FSM rewrite. Add settings: MAX_JOIN_THRESHOLD, DECIDE_INTERVAL, JOIN_ENTER_SCORE/LEAVE_SCORE, JOIN_DWELL/LEAVE_DWELL, MANUAL_HOLD_DURATION.

Tests: new tests/test_personality.py (determinism, decorrelation from appearance hash, range); test_character/test_world for autonomous join/leave with hysteresis (no per-frame flip-flop) and that a chat command isn't immediately undone. `uv run --project . pytest` green. Touch only the listed files.
