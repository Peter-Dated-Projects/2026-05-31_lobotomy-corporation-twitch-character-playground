## Goal
Find 3-10 candidate sprites for the 'robot body' speak feature from the Sephirah + Ayin & Myo asset drop. Output a ranked shortlist to `.charm/kb/robot-sprite-candidates.md`.

## Directory
`/Users/petthepotat/Downloads/drive-download-20260606T214433Z-3-001/Sephirah + Ayin & Myo/`

Subdirs: Angela, Ayin, Benjamin, Binah, Carmen, Chesed, Gebura, Hod, Malkuth, Myo, Netzach, Rabbits, Tiphereth, Yesod

## What to look for
Each character folder has multiple PNGs. We know Chesed has `ChesedRobot-resources.assets-1402.png`. We want:
- Any file with 'Robot' in the name (these are the in-game robot body sprites, highest priority)
- Full-body standing/animated sprites (NOT portraits, NOT CG cutscenes -- portrait/CG files tend to have 'Portrait' or 'CG' or 'Past' in the name)
- Sprites that look like they could be the body of a character speaking on screen (humanoid, standing pose, not a close-up face)

## How to scan
- `ls` each character subdirectory and read the filenames -- DO NOT open every image
- For each character, note which files look like: robot variant, full-body sprite, portrait (to exclude), or CG art (to exclude)
- For the top candidates (Robot files + clean standing body files), view the image using the Read tool to confirm it is a usable sprite (not a portrait or abstract art)
- Limit image reads to at most 2-3 per character, prioritizing Robot-named files first

## Output format for `.charm/kb/robot-sprite-candidates.md`
```
# Robot Sprite Candidates

## Tier 1 - Explicit Robot variants
- CharacterName: /full/path/to/file.png -- one-line description of what you saw

## Tier 2 - Full-body standing sprites (usable as robot body)
- CharacterName: /full/path/to/file.png -- one-line description

## Excluded
- Brief note on what was skipped (portraits, CG, bosses, etc.) and why
```
